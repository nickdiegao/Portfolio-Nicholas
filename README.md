Portfólio pessoal desenvolvido em **React**, apresentando trajetória profissional, stack técnica, projetos e informações de contato.

## ✨ Visão Geral

Este portfólio foi criado para apresentar de forma clara e elegante o perfil de Nicholas Diego de Lira Veloso, desenvolvedor de software com foco em **Backend**, **APIs RESTful** e **Cloud AWS**.

O design segue uma estética **dark sofisticada** com paleta Terracota & Bege, tipografia editorial e micro-animações de scroll reveal.

## ⚙️ Como Rodar Localmente

### Pré-requisitos

- Node.js 18+
- npm ou yarn

### Passos

```bash
# 1. Clone o repositório
git clone https://github.com/nickdiegao/portfolio.git
cd portfolio

# 2. Instale as dependências
npm install

# 3. Inicie o servidor de desenvolvimento
npm run dev
# ou
npm start
```

Acesse em: `http://localhost:3000`

---

## 🌐 Deploy na Vercel

A forma mais rápida de publicar:

```bash
# Instale a CLI da Vercel
npm i -g vercel

# Faça o deploy
vercel
```

Ou conecte o repositório diretamente em [vercel.com](https://vercel.com) para deploy automático a cada `git push`.
